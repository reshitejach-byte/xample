# Portfolio Website

A simple, responsive portfolio website scaffold built with HTML, CSS, and JavaScript.

## Files

- `index.html` — the main portfolio page.
- `styles.css` — responsive layout, dark mode support, and UI styling.
- `script.js` — theme toggle and local storage persistence.

## Usage

1. Open `index.html` in your browser.
2. Edit the content to replace `Your Name`, project details, and contact information.

## Notes

- The project is intentionally minimal and works without a build step.
- Update the `mailto:` link in `index.html` for your email.
- Optionally add real project links or images to personalize the portfolio.
